NOTES
-----
Parts used in JWE Compact Serialization are:
- Protected header
- Content encryption key (CEK) is the symmetric key that protects the payload, encrypted using RSA-OAEP
- Initialization vector
- Ciphertext / payload is encrypted using the above symmetric key
- Authentication tag

Asymmetric encryption is used to give the key to the recipient (the website)
OAEP specifies a type of padding
Symmetric encryption is used for the main part of the encryption, called authenticated encryption
Resulting bytes are base64url encoded
A random IV is created
